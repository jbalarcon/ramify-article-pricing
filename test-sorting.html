Sorting Functionality Test

Test Checklist

  1. Load the application with sample data
  2. Click on writer table headers to sort:
    • Click "Rédacteur" - should sort alphabetically
    • Click "Articles" - should sort by article count
    • Click "Mots Moyens" - should sort by average word count
    • Click "CV" - should sort by coefficient of variation
    • Click any cost column - should sort numerically
  3. Click on article table headers to sort:
    • Click "URL" - should sort alphabetically
    • Click "Rédacteur" - should sort by writer name
    • Click "Date" - should sort chronologically
    • Click "Nombre de Mots" - should sort numerically
    • Click cost columns - should sort by amounts
  4. Verify arrow indicators:
    • Up arrow (▲) for ascending sort
    • Down arrow (▼) for descending sort
    • Click same header twice to toggle direction
  5. Test search with sorting:
    • Sort a column
    • Use search to filter results
    • Clear search - sorting should persist
