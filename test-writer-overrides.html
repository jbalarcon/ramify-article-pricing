Test Writer-Specific Pricing Overrides

Issue Fixed

✓ Writer-specific bonus percentages are now captured correctly

Previously, the bonus percentage for writer overrides was always set to 0. This has been fixed.

How to Test Writer-Specific Pricing

Step 1: Load Sample Data

Upload the sample-data.csv file which contains 4 writers:

  • Jean Dupont
  • Marie Martin
  • Pierre Durand
  • Sophie Bernard

Step 2: Open Configuration Modal

Click "Ouvrir la configuration" button

Step 3: Go to "Nouvelle Tarification" Tab

Keep the global default as is (PW with 0.13€/word)

Step 4: Customize Individual Writers

In the "Personnalisation par rédacteur" section:

  • Jean Dupont: Check the box, select "Prix fixe" (FP), set to 150€
  • Marie Martin: Check the box, keep "Tarif au volume" (PW), change rate to 0.15€/word
  • Pierre Durand: Leave unchecked (will use global default of 0.13€/word)
  • Sophie Bernard: Check the box, select "Prix plafonné" (CO), set Fixed: 200€, Cap: 1700 words, Overage: 0.10€

Step 5: Apply Configuration

Click "Appliquer la configuration" button

Step 6: Verify Results

Check the console (F12) to see:

Writers with custom pricing: ['Jean Dupont', 'Marie Martin', 'Sophie Bernard']

The median costs should now be different between Référence and Simulation!

Expected Results

  • Référence median: Should be based on all writers using 0.13€/word
  • Simulation median: Should be different due to mixed pricing models
  • Per-writer costs: Should vary based on their specific configurations

Example calculations:

  • Jean Dupont with FP 150€: All articles = 150€ regardless of length
  • Marie Martin with PW 0.15€: 1000 words = 150€ (vs 130€ in reference)
  • Pierre Durand with default: 1000 words = 130€ (same as reference)
  • Sophie Bernard with CO: 1500 words = 200€, 2000 words = 230€

Console Verification

Open the browser console and look for:

Configuration comparison: Global baseline config: {model: 'PW', params: {rate: 0.13}, bonusPercent: 0} Global simulation config: {model: 'PW', params: {rate: 0.13}, bonusPercent: 0} Writer overrides: { 'Jean Dupont': {simulation: {model: 'FP', params: {amount: 150}, bonusPercent: 0}}, 'Marie Martin': {simulation: {model: 'PW', params: {rate: 0.15}, bonusPercent: 0}}, 'Sophie Bernard': {simulation: {model: 'CO', params: {...}, bonusPercent: 0}} }
