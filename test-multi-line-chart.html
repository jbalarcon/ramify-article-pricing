Test Multi-Line Word Count Evolution Chart

New Features

✓ Multiple Lines on Chart

  • Blue thick line: Global average (all writers combined)
  • Colored lines: Individual writer averages
  • Solid lines for even-indexed writers
  • Dashed lines for odd-indexed writers (for visual distinction)

✓ Handles Different Start Dates

  • Writers who haven't started yet show as null (no line)
  • Lines connect across missing months with spanGaps: true
  • Each writer's line starts when they have their first article

✓ Interactive Features

  • Hover over any point to see exact values
  • Click on legend items to show/hide specific lines
  • Tooltip shows all values for a given month

Testing Steps

  1. Load the application below
  2. Upload the sample-data.csv file
  3. Look at the "Évolution du Nombre de Mots" chart
  4. Verify you see multiple lines (1 global + 4 writer lines)
  5. Hover over different points to see the tooltip
  6. Click on legend items to toggle line visibility
  7. Notice how lines start at different months if writers joined later

What to Look For

  • Global trend line (blue) should be thicker and always visible
  • Each writer has their own color and line style
  • Legend shows all writers with their colors
  • Tooltip displays values for all visible lines at once
  • Lines properly handle months with no data
